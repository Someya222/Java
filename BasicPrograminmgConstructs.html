Write a Java program to print the sum of two numbers


                import java.io.BufferedReader;
                import java.io.InputStreamReader;
                import java.io.IOException;
                
                public class SumOfTwoNumbers {
                    public static void main(String[] args) throws IOException {
                        BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
                
                        System.out.print("Enter the first number: ");
                        int num1 = Integer.parseInt(reader.readLine());
                        System.out.print("Enter the second number: ");
                        int num2 = Integer.parseInt(reader.readLine());
                
                        int sum = num1 + num2;
                
                        System.out.println("Sum: " + sum);
                    }
                }

Output


                Enter the first number: 22
                Enter the second number: 11
                Sum: 33

Write a Java program to print the sum, multiply, subtract, divide and remainder of two numbers


                import java.io.BufferedReader;
                import java.io.InputStreamReader;
                import java.io.IOException;
                
                public class BasicOperations {
                    public static void main(String[] args) throws IOException {
                        BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
                
                        System.out.print("Enter the first number: ");
                        int num1 = Integer.parseInt(reader.readLine());
                        System.out.print("Enter the second number: ");
                        int num2 = Integer.parseInt(reader.readLine());
                
                        int sum = num1 + num2;
                        int product = num1 * num2;
                        int difference = num1 - num2;
                        double quotient = (double) num1 / num2;
                        int remainder = num1 % num2;
                
                        System.out.println("Sum: " + sum);
                        System.out.println("Product: " + product);
                        System.out.println("Difference: " + difference);
                        System.out.println("Quotient: " + quotient);
                        System.out.println("Remainder: " + remainder);
                    }
                }

Output


                Enter the first number: 4
                Enter the second number: 2
                Sum: 6
                Product: 8
                Difference: 2
                Quotient: 2.0
                Remainder: 0

Write a Java program that takes five numbers as input to calculate and print the average of the numbers


                import java.io.BufferedReader;
                import java.io.InputStreamReader;
                import java.io.IOException;
                
                public class AverageOfFiveNumbers {
                    public static void main(String[] args) throws IOException {
                        BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
                
                        System.out.print("Enter the first number: ");
                        double num1 = Double.parseDouble(reader.readLine());
                        System.out.print("Enter the second number: ");
                        double num2 = Double.parseDouble(reader.readLine());
                        System.out.print("Enter the third number: ");
                        double num3 = Double.parseDouble(reader.readLine());
                        System.out.print("Enter the fourth number: ");
                        double num4 = Double.parseDouble(reader.readLine());
                        System.out.print("Enter the fifth number: ");
                        double num5 = Double.parseDouble(reader.readLine());
                
                        double sum = num1 + num2 + num3 + num4 + num5;
                        double average = sum / 5;
                
                        System.out.println("Average: " + average);
                    }
                }

Output


                Enter the first number: 3
                Enter the second number: 6
                Enter the third number: 2
                Enter the fourth number: 1
                Enter the fifth number: 7
                Average: 3.8

Write a Java program to swap two variables


                import java.io.BufferedReader;
                import java.io.InputStreamReader;
                import java.io.IOException;
                
                public class SwapVariables {
                    public static void main(String[] args) throws IOException {
                        BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
                
                        System.out.print("Enter the first variable: ");
                        int a = Integer.parseInt(reader.readLine());
                        System.out.print("Enter the second variable: ");
                        int b = Integer.parseInt(reader.readLine());
                
                        int temp = a;
                        a = b;
                        b = temp;
                
                        System.out.println("After swapping:");
                        System.out.println("First variable: " + a);
                        System.out.println("Second variable: " + b);
                    }
                }

Output


                Enter the first variable: 56
                Enter the second variable: 89
                After swapping:
                First variable: 89
                Second variable: 56
